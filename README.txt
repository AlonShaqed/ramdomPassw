##randomPassw.py##

#Ejecución#
-Instalar python 3.7.x. Se puede bajar desde la página oficial para Windows (https://www.python.org/).
-Descargar la libreria de appJar para la interfaz. Esta disponible en: http://appjar.info
-Si utiliza Python en Mac o Linux, refiera a http://appjar.info/Install/
-Una vez instalado, solo basta darle doble click al archivo 'randomPassw.py' o ejecute desde terminal.

*Nota: Si appjar no esta disponible, el codigo no va a funcionar

#Uso#
El programa recibe una cadena de caracteres. Si los caracteres son menos de cuatro, se insertan tal cual, si son mas, se seleccionaran cuatro al azar. El programa genera cuatro numeros al azar por si solo.
